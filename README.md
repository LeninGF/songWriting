# Song Collection

This repository contains a collection of original songs written in Org mode format by me Lenin Falconi.

## Files

- **AlzaTuVoz.org** – Lyrics and Chords Original Work
- **FascistProblem.org** – Original Lyrics and Chords are a cover of Drinking Problem by Midlan
- **dejenLaPauta.org** – Original Lyrics. Chords sequence is inspired in The Beatles Let it Be but as a punk song. This was tried with Suno AI application.

## Format

Each song is stored as an `.org` file with:
- Lyrics structured in verses, choruses, and bridges
- Metadata (title, author, date, etc.)
- Backup versions (`.org~`) are automatically created

## Usage

Open any `.org` file in Emacs or a text editor to view/edit the song. The Org mode syntax allows for easy formatting and organization.
